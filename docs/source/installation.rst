Installation
============

Install from source (recommended for users)
-------------------------------------------

.. code-block:: bash

   git clone https://github.com/w-ruizhe/spectralmap.git
   cd spectralmap
   pip install .

or

.. code-block:: bash

   pip install spectralmap

Development install (optional)
------------------------------

.. code-block:: bash

   pip install -e ".[dev,docs]"
